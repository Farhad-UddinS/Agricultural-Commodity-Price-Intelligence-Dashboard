# Agricultural Commodity Price Intelligence Dashboard

## Project Overview

Developed an interactive Excel dashboard to analyze historical agricultural commodity prices and support procurement, investment, and risk-management decisions.

The dashboard helps businesses:
- identify stable commodities for long-term contracts
- monitor highly volatile commodities
- analyze historical price trends
- support data-driven sourcing decisions

---

# Business Problem

Commodity-driven businesses often struggle with:
- unstable raw material pricing
- procurement risk
- poor cost forecasting
- lack of historical pricing insights

This project solves these challenges by transforming raw commodity pricing data into an interactive business intelligence dashboard for trend, volatility, and stability analysis.

---

# Data Preparation Using Power Query

The raw dataset was cleaned and transformed using Power Query.

### Data Preparation Steps
- Removed incomplete/future records
- Cleaned percentage columns and removed errors
- Converted columns to proper decimal and date formats
- Standardized data types
- Unpivoted wide-format commodity columns into analysis-ready structure
- Built a clean dataset optimized for Pivot Tables and dashboarding

---

- Interactive slicers and timeline filters
- Commodity trend analysis
